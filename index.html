About

Hello! I'm Clem, a Junior Developer

I recently graduated from the General Assembly Software Engineering Immersive bootcamp and I'm currently looking for my first Junior Developer role.

My interest in programming started after reading an article about Python and started with 'Learn Python the Hard Way' which gave me a foundation to object-oriented programming and the command line. My favorite project was a text-adventure game for the terminal to hone my understanding of learning concepts. I made it absurd and laughed the whole way through and thus began my coding path. From there I wanted code full time and so enrolled in a bootcamp to accelerate my learning process.

During the bootcamp I created and collaborated on 4 projects: a Canvas JavaScript game, a front end consuming the Spotify podcast API, a full-stack MERN e-commerce store and a React, Django and PostgreSQL exercise dashboard.

My instructors describe me as ambitious and unafraid of working with new concepts and technologies (or lack of coding stage-fright). I would describe myself as confident and driven, able to work well as an individual contributor and as part of a team and with strong communication skills.

My next project will be using Firebase and TypeScript to develop an app using the Test Driven Development method and Jest testing suite.

Outside of coding I follow Tampa Bay Lightning (NHL team) and play and record guitar. I also enjoy swimming, going on bike rides or rollerblading.
